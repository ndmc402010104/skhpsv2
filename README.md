# skhpsv2

新光整形外科室系統 v2 架構實驗專案。

## 第一階段定位

本階段先建立 CSS / UI Design System 與 Portal shell。

目前不連接 Apps Script、不搬移 QR / Dressing / HIS / legacy 功能。

## 第一階段目標

- 建立完整 CSS 分層架構
- 建立 Portal 前台外觀測試頁
- 建立 Admin 後台外觀測試頁
- 建立 UI Design System 測試頁
- 先驗證 footer、button、card、form、table、mobile responsive 等共用樣式

## 頁面

- index.html：前台 Portal 外觀測試
- admin.html：後台入口外觀測試

## 第一階段禁止事項

- 不連接 Apps Script
- 不搬 QR 功能
- 不搬 Dressing 功能
- 不搬 HIS 功能
- 不修改舊專案
