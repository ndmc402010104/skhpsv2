院內系統

18:00 跳台倒數自動送出

在院內開刀系統的跳台申請頁面執行；預設篩選「住院手術 + 三房」，可自行勾選，時間到（18:00）會自動送出並按「是」。 兩種用法：把下面按鈕拖到瀏覽器書籤列，之後在該頁面點一下即可；或複製整段程式碼，在該頁面按 F12 開 Console 貼上執行。

手機版

在手機上使用（userscript）

手機沒有 F12 Console 可以貼、書籤在院內系統的內容安全政策（CSP）下也不可靠，所以手機要靠「userscript 管理器」——它把腳本跑在獨立沙盒、不受頁面 CSP 限制。裝好之後點一次安裝連結即可；之後在同一個瀏覽器用下面的按鈕進院內系統，跳台頁面左上角就會自動浮出「▶ 18:00 跳台」按鈕。

📲 安裝手機腳本（.user.js） 🏥 開啟醫院系統快速登入

（安裝連結要在「已經裝好 userscript 管理器的瀏覽器」裡打開才會跳出安裝視窗，直接點只會看到純文字程式碼、那是正常的。「開啟醫院系統快速登入」會在你目前這個瀏覽器開，所以請在裝了腳本的那個瀏覽器裡點它。）

Android 安裝步驟（推薦 Via 瀏覽器）

📥 下載 Via 瀏覽器
  1. 沒有 Via 的話，先點上面「📥 下載 Via 瀏覽器」到 Google Play 安裝（很小、內建 userscript，不用另外裝擴充）。
  2. 用 Via 打開這一頁，點「📲 安裝手機腳本」，Via 會問要不要加入腳本，按確定。
  3. 接著在 Via 裡點「🏥 開啟醫院系統快速登入」，登入進到跳台（預約手術病人列表）頁面。
  4. 畫面左上角浮出「▶ 18:00 跳台」按鈕，點它即可（跟電腦上跑起來一模一樣）。

小提醒：在 Via 開啟醫院快速登入後，用 Via 選單的「加到主畫面」做一顆桌面捷徑，以後每天一點就直接用 Via 進院內系統，最快。裝好之後我再更新腳本，Via 會自動抓新版、不用重裝。（也可以用 Firefox ＋ Tampermonkey；先前提過的 Kiwi 較難操作，不建議。）

iPhone / iPad 安裝步驟

  1. App Store 安裝免費的「Userscripts」App。
  2. 到 iOS 設定 → Safari → 擴充功能 → 開啟「Userscripts」並允許存取網站。
  3. 用 Safari 打開這一頁，點「📲 安裝手機腳本」；若沒直接跳安裝，改用 Safari 分享／Userscripts App 匯入這個 .user.js 連結。
  4. 接著在 Safari 裡點「🏥 開啟醫院系統快速登入」，登入進跳台頁面，左上角浮出「▶ 18:00 跳台」按鈕，點它即可。
