# legacy-tools

這個資料夾暫存舊系統 Apps Script / clasp / push 流程工具。

目前 skhpsv2 第一階段是純 GitHub 前端 CSS / UI Design System，不應直接使用這裡的 push.ps1 部署。

注意：
- 不要把舊 .clasp.json 放到 skhpsv2 根目錄。
- 未來建立 skhps-api 時，應重新建立新的 Apps Script 專案與新的 .clasp.json。
- push.ps1 / clasp-tools.ps1 / update-version.ps1 之後要重寫成 skhps-api 專用版本。
